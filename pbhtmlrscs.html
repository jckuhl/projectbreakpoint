HTML and CSS

Here you can find resources for HTML and CSS

Mozilla Developer Network

Mozilla Developer Network: HTML

Mozilla Developer Network: CSS

The Mozilla Developer Network is your goto reference for all things web development. They offer indepth references and tutorials for you to use.

Word Wide Web Consortium

W3 Consortium

HTML Validator

CSS Validator

The W3C is the official source for HTML and CSS standards. They also provide validators that can go through your HTML and CSS and ensure they meet standards. Note that the W3C is not in any way related to W3Schools.

CSS Tricks

CSS Tricks

A resource for advanced HTML and CSS tricks to add to your website.

Bootstrap

Bootstrap

Bootstrap is a CSS library that makes layouts easier. Its library is extensive and powerful. A developer can, simply by applying bootstrap classes, quickly design and layout a webpage.

Google Fonts

Google Fonts

Google Fonts is a major source for fonts. It provides multiple weights and measures the performance hit such a font will have on your webpage, allowing you to decide wisely between style and performance. It also provides the markup necessary for linking and using the fonts in your project.
