JavaScript Resources

Here you can find resources for learning JavaScript and finding resources and documentation, as well as places you can play around.

JavaScript on the MDN

JavaScript - Mozilla Developer Network

As always, the Mozilla Developer Network is an amazing resource for indepth research on JavaScript. They provide documentation for the hundreds of different methods, explanations and examples on the syntax and tutorials to help teach you everything from the basics to classes and closures. The MDN is my first go-to whenever I have a JavaScript problem to solve.

jQuery API

jQuery API

Having problems with your jQuery? The official API gives you documentation for all the various functions and methods the jQuery library brings you. You're also only a few clicks away from the jQuery UI API and links to CDNs and downloads to get jQuery installed on your project.

JSFiddle

JSFiddle

JSFiddle is a site where you can write and test your own code. It's a great resource for practice, experimentation and development, giving you a place to work and get immediate results without having to constantly make new files on your harddrive.

CodePen

Code Pen

Similar to JSFiddle, CodePen is a place to test web code. You can also browse the various pens other programmers have authored. Not only is it suited for JavaScript, it is also suitable for testing CSS and SVG code.
